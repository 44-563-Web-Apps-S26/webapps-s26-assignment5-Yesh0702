Responsiveness Observation

Which row breaks first? The tree names row (the second row) breaks first.

Explanation: The second row uses the col-md class, which triggers a vertical stack (break) at the "medium" breakpoint of 768px. The first row uses col-sm-2, which maintains its horizontal structure until the "small" breakpoint of 576px.
